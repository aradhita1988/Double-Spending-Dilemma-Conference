# Double-Spending-Dilemma-Conference
Multichain Delay Simulation

A **realistic multichain transaction delay and double-spending attack simulation** using **Ganache**, **Web3.js**, and a simple **Solidity smart contract** (`DelaySim.sol`).

This simulation evaluates how increasing **transaction delay on Chain A** affects **double-spending probability** and **detection time** across interoperable blockchains.
